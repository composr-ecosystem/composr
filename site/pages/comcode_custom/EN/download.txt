<h3 class="colrBl"><strong>Git read access</strong></h3>
<strong> </strong>

<div class="sept"><strong>&nbsp;</strong></div>
<strong> </strong>

<p><strong><a href="http://git-scm.com/">Git</a> is the revision control system we use to collaborate on Composr.</strong></p>
<strong> </strong>

<p><strong>You can access the repository on <a href="{$COMPOSR_REPOS_URL*}">GitLab</a>. Version 11 is on the &quot;v11&quot; branch.</strong></p>
<strong> </strong>

<p><strong>We often commit fixes to Git before patch releases come out.</strong></p>
<strong> </strong>

<p><strong>Be aware that the Git repository also has many non-bundled addons in it. We develop them in there, to keep them up-to-date. There is a script called [tt]delete_alien_files.php[/tt] which will erase them. You need to first delete the [tt]sources_custom/hooks/systems/addon_registry/*.php[/tt] files of any addons you don&#39;t want, then run the script, then run the commands the script says (after reviewing), then install.</strong></p>
